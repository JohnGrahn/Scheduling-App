Title

SchedulingApp is a java application which allows users to manage appoinments with customers 
across several countries and timezones. SchedulingApp also allows users to generate reports on information
stored within the system.

Running Program
JDK and JavaFX are required to run this application.
Upon launch the user will find themselves on a login screen. A username and password that match records in
the database will be required.
